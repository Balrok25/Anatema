# NORMAS DE ACTUACIÓN PARA LA CREACIÓN DEL COMPENDIUM MURI

## 1. PRINCIPIO DE FUENTE VERIFICABLE
- Todo conocimiento debe ser rastreable hasta fuentes históricas reales (grimorios, tratados de demonología, bestiarios, manuales de exorcismo).
- No se inventan conceptos, rituales ni entidades no documentados.

## 2. PRINCIPIO DE AMALGAMA, NO DE CITA
- No se menciona explícitamente de dónde sale cada cosa.
- Información de múltiples fuentes se presenta como un sistema unificado.

## 3. PRINCIPIO DE DESPOJO
- Se elimina todo "maquillaje" histórico: lenguaje arcaico, supersticiones de la época, referencias religiosas específicas.
- Se traducen conceptos antiguos al lenguaje moderno y directo.

## 4. PRINCIPIO DE EXHAUSTIVIDAD
- Cuando se desarrolla un tema, se hace hasta el detalle más fino.
- Cada subcapítulo responde: qué es, por qué, cómo, cuándo, dónde, riesgos, variantes.

## 5. PRINCIPIO DE PRACTICIDAD
- La información debe ser usable.
- Cada instrucción es clara, precisa y reproducible.
- Se incluyen detalles prácticos: materiales exactos, dimensiones, tiempos, gestos.

## 6. PRINCIPIO DE COHERENCIA INTERNA
- El manual es coherente dentro de sí mismo.
- Términos y taxonomías son uniformes en todo el documento.
- Sin contradicciones internas.

## 7. PRINCIPIO DE ADVERTENCIA CONSTANTE
- Cada capítulo con riesgo incluye advertencias explícitas.
- Se incluyen consecuencias: corto plazo, largo plazo, personales, colectivas.

## 8. PRINCIPIO DE ESTRUCTURA CLARA
- Cada capítulo tiene estructura consistente:
  - Definición
  - Subdivisiones temáticas
  - Explicación exhaustiva
  - Variantes o excepciones
  - Riesgos y consideraciones finales

## 9. PRINCIPIO DE LENGUAJE NEUTRAL Y DIRECTO
- Lenguaje de manual técnico: claro, preciso, sin ambigüedades.
- Sin lenguaje místico innecesario, florido o poético.

## 10. PRINCIPIO DE SÍNTESIS DE VARIANTES
- Cuando tradiciones diferentes dicen cosas sobre el mismo tema, se sintetizan en un sistema coherente.
- Si hay contradicciones irreconciliables, se presentan ambas opciones pero se indica cuál es más efectiva.

## 11. PRINCIPIO DE NOMENCLATURA CONSISTENTE
- Se usan términos definidos y consistentes:
  - "Círculo de protección" (no variaciones)
  - "Punto de manifestación"
  - "Sello" (no sigilo, firma, marca)
  - "Entidad" (no espíritu, demonio, cosa)

## 12. PRINCIPIO DE JERARQUÍA DE INFORMACIÓN
- Información crítica se presenta primero.
- Detalles secundarios vienen después.
- Advertencias se repiten en contextos críticos.

## 13. PRINCIPIO DE VERIFICACIÓN ANTE LA DUDA
- Si hay incertidumbre sobre si algo es histórico o inventado, se investiga.
- Si no se puede verificar, se omite.
- No se adivina.

## 14. PRINCIPIO DE ACTUALIZACIÓN CONTINUA
- El manual es vivo, no cerrado.
- Se integran nuevas informaciones de fuentes verificadas.

## 15. PRINCIPIO DE NO FICCIÓN
- Absolutamente prohibido:
  - Inventar anécdotas de "cazadores"
  - Crear historias de casos resueltos
  - Añadir narrativas o elementos dramáticos
  - Personificar el conocimiento

## 16. PRINCIPIO DE CONTEXTO LIMITADO
- No se incluye "por qué la gente creía esto en la Edad Media"
- No se incluye historia de las creencias
- Se incluye cómo funciona, qué hacer, cuáles son los pasos
